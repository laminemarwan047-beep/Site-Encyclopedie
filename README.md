# ARCHETYPE - Encyclopédie d'Architecture

##  Présentation du projet
Projet académique réalisé dans le cadre du semestre 2 de la formation PREPA DEV. 
Le site est une encyclopédie thématique dédiée à l'architecture moderne et contemporaine.

## Structure du site

Le projet est composé de 3 pages principales :

index.html : Page d’accueil (présentation + catégories)
articles.html : Liste des articles (cartes avec images )
detail.html : Page de détail d’un article
